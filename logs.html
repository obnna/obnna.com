Logs Analysis

Project

This project is running off a virtual machine based on Ubuntu 14 with postgresql pre-installed. For windows users, all software required to run the project are as follows: 1. Git or project zip (https://github.com/obnna/fullstack-nanodegree-vm.git) 2. Vagrant 3. Oracle Virtual Box Vagrant is a useful tool that allows running a virtual machine through windows command prompt. It is ightweight, and can run for hours without little to no impact on PC performance. I used vagrant 1.9.5 for Windows .

Get oracle virtual box, which is needed for vagrant to work. Clone this repo using git or any terminal of your choice. Or you have the option of downloading the project as a zip file from Github. After installing, open git bash, cmd, or even powershell which I highly recommend.

In your chosen directory, type "git clone", followed by the url of the project. For this project, type git clone https://github.com/obinna1/fullstack-nanodegree-vm.git. Now that the files have been cloned onto your computer, you will use vagrant to install the virtual machine. To do this, open command prompt and navigate to the repo you just cloned. For example, if you cloned the repo into your documents directory, type cd documents/fullstack-nanodegree-vm.

In the folder type "vagrant up". This command will download the configured virtual machine unto your PC. Beware, you may have issues getting past this stage. I had to go into BIOS and enable Intel Virtualization for the VM to install properly. To get into BIOS, click f12 or f2 repeadedly as you start up your PC. After vagrant has installed the VM, you will need to login using the command "vagrant ssh" To access shared directory, cd to /vagrant.

At this point, use ls (list) to confirm you have folders named newsdata, forum and tournaments. For the entirety of this project, you will use the newsdata folder. Due to Github's max file size of 100mb, I intentionally left out the newsdata.sql, which you will have to download and place in the newsdata folder. Here is a link to the file --https://goo.gl/ OtdXKS After moving the downloaded files to the newsdata folder, navigate to that folder using in your VM and type "psql -d news -f newsdata.sql". This will load the data into the database.

This project connects to a database in a virtual machine, and utilize complex SQL queries to analyze log data from a huge database, then print out answers to pressing customer questions.

Now this project is asking for the results of three questions which are as follows:

1. What are the most popular three articles of all time?

2. Who are the most popular article authors of all time?

3. On which days did more than 1% of requests lead to errors?

To obtain these answers, you will type "python newsdb.py" in the newsdata directory of your virtual machine
